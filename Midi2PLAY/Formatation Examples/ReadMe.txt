Here are formatation example files. To be used with the Instructions. 

"mml_before.txt" --> The "mml" file as came out from 3ML Editor, that is, before the formatation. This will not work in Midi2PLAY.

"mml_with_duplicated_bar.txt"  --> Almost ready. Still need to erase (manually) the duplicated measure marks (the SECOND "/*M2*/"in [Channel2]). 
Use this file (as it is) in Midi2PLAY and it will print the duplicated marks in "temp.txt.

"mml_ready.txt"  --> "mml" after formatation, ready to use in "midi2play".

"temp.txt"  --> Where the duplication indications are stored.

To test some of this files on "Midi2PLAY", save it as "mml.txt" and move it to the root folder.

-----------------------------Leia-me (PT-BR)---------------------------------


"mml_before.txt" --> O arquivo "mml" exportado pelo 3ML Editor, ou seja, antes da formatação. Este arquivo não funcionará no Midi2PLAY.

"mml_with_duplicated_bar.txt"  --> Quase pronto. Ainda precisa apagar as marcações duplicadas. No caso o segundo "/*M 2*/"  no "[Channel2]".
 Use este arquivo (do jeito que está) no Midi2PLAY e ele indicará as duplicações em "temp.txt".

"mml_ready.txt"  --> "mml" completamente formatado, isto é, pronto para ser convertido no "midi2play".

"temp.txt"  --> Onde são indicadas as marcações duplicadas. Utilize-o para encontrá-las.

Para testar alguns destes arquivos no Midi2PLAY, salve-os como "mml.txt" e os mova para a pasta raiz da aplicação.
